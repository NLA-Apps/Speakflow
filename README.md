# SpeakFlow 🎙️ — מאמן האנגלית האישי שלך

אפליקציית ווב לתרגול אנגלית בשיחה קולית עם בוט AI — מותאמת למחשב, לאייפון ולאנדרואיד.

## פיצ'רים

- 🎤 **שיחה קולית** — מדברים באנגלית, הבוט מבין, עונה ומקריא בקול
- ✨ **תובנות בזמן אמת** — הערכת רמה (A1–C2), ספירת מילים, מילים חדשות, תיקוני דקדוק וטיפים בעברית
- 🌐 **תרגום בלחיצה** — לוחצים על כל מילה בשיחה ומקבלים תרגום; אפשר לתרגם משפט שלם בכפתור
- 🔖 **שמירה ללימוד** — מילים ומשפטים נשמרים והופכים לכרטיסיות
- 🃏 **כרטיסיות חזרה (Spaced Repetition)** — חוזרות בדיוק כשעומדים לשכוח
- 🗣️ **תרגול הגייה** — קוראים משפט בקול ורואים אילו מילים זוהו נכון + ציון
- 🎭 **תרחישי משחק תפקידים** — ראיון עבודה, מסעדה, שדה תעופה, קניות, טלפון, רופא
- 📈 **גרף התקדמות** — מילים ליום ב-14 הימים האחרונים, שיא יומי, ימי תרגול
- 🔥 **רצף ימים ויעד יומי** — מוגדר בהגדרות, עם טבעת התקדמות
- 🔊 **בחירת קול** — בחירת הקול הטבעי ביותר במכשיר + מהירות דיבור
- 📱 **PWA** — באנדרואיד אפשר "להוסיף למסך הבית" ולקבל אפליקציה

## איך מריצים

**במחשב:** לחיצה כפולה על `START.bat` → נפתח דפדפן ב-http://localhost:8765

**בטלפון (באותו WiFi):** הרץ את `START.bat` במחשב, מצא את כתובת ה-IP של המחשב
(`ipconfig` → IPv4), ובטלפון גלוש ל-`http://<IP>:8765`.
שים לב: מיקרופון בדפדפן דורש HTTPS או localhost — בטלפון המיקרופון יעבוד רק אם מגישים
דרך HTTPS (למשל דיפלוי חינמי ל-Netlify/Vercel/GitHub Pages — מומלץ לשלב הבא!).

## חיבור ה-API

1. נרשמים ב-https://console.anthropic.com ויוצרים API key
2. באפליקציה: ⚙️ הגדרות → מדביקים את המפתח → שמירה
3. בלי מפתח — מצב דמו (הכל עובד חוץ מתשובות אמיתיות של הבוט)

## מבנה הפרויקט

| קובץ | תפקיד |
|---|---|
| `index.html` | מבנה הממשק — 4 מסכים: שיחה, תובנות, תרגול, התקדמות |
| `css/style.css` | עיצוב רספונסיבי (ניווט תחתון במובייל, פאנל צד בדסקטופ) |
| `js/config.js` | מודל, system prompt, תרחישים, משפטי הגייה |
| `js/storage.js` | שמירה מקומית של הכל |
| `js/translate.js` | תרגום EN→HE (Google gtx + גיבוי MyMemory, עם cache) |
| `js/srs.js` | מנוע כרטיסיות — תזמון חזרה בסגנון SM-2 |
| `js/progress.js` | סטטיסטיקות יומיות, רצף, גרף SVG |
| `js/insights.js` | פאנל תובנות + טבעת יעד יומי |
| `js/speech.js` | זיהוי דיבור + הקראה + בחירת קול |
| `js/api.js` | Claude API (structured output) + מצב דמו + תרחישים |
| `js/app.js` | הלוגיקה הראשית |
